Structure
=========

include:
	 mymex.h that implements all necessary functions generated by 
	 the tool

config:
	Two tamplates that are used by the mex file generator. Don't edit
	it unless you know what are you doing. 

tests:
	Test input files for testing amg

usr/share:
	useful mexfunctions can be shared here, more useful functions can 
	be added from a version to another    


* If you change the structure of this software, update the target 
  cleanall in the makefile

* Any binary/object file that exist any where should be cleaned by 
  the target clean_all in the makefile
